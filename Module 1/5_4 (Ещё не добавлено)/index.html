Главная страница

- Измените цвет фона страницы с помощью свойства background-color.- Измените цвет текста с помощью свойства color.
- Измените шрифт и размер текста с помощью свойства font-family и font-size.
- Измените отступы и границы элементов с помощью свойств margin и border.
- Измените выравнивание текста и элементов с помощью свойств text-align и float.
- Измените внешний вид ссылок, кнопок и других интерактивных элементов с помощью псевдоклассов (:hover, :active, :visited).
- Измените внешний вид заголовков с помощью свойств font-weight, text-transform или других свойств.
- Добавьте комментарии в файл .css, чтобы описать, какие стили относятся к каким элементам.
